# TurfSlot

Standalone Vite + React project.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Notes

- Path alias `@` points to `src`.
- Local data is handled by [src/api/client.js](src/api/client.js) using an in-memory store.
