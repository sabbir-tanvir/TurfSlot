# TurfSlot

Starter setup for a Vite project with ESLint and Tailwind CSS config files.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Notes

- The app entry HTML is `index.html`.
- The page expects a module entry at `src/main.js`.
- Path alias `@/` maps to `src/` via `jsconfig.json`.
